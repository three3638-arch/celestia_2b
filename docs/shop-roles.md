# 2C 后台角色权限矩阵

| 操作 | SHOP_EDITOR | SHOP_ADMIN |
|------|:-----------:|:----------:|
| 查看商品/询价 | ✓ | ✓ |
| 查看品类/留言 | ✓（只读） | ✓ |
| 创建/编辑品类 | ✗ | ✓ |
| 删除品类 | ✗ | ✓ |
| 创建/编辑商品（含 SKU 定价） | ✗ | ✓ |
| 商品上下架（status） | ✗ | ✓ |
| 上传/删除商品图片 | ✓ | ✓ |
| 更新询价/留言状态 | ✗ | ✓ |

实现约定：

- `requireShopUser()`：任意已登录 2C 后台用户（读 + 图片管理）
- `requireShopAdmin()`：品类写操作、定价、商品 CRUD、上下架、删除、状态流转

**导航裁剪**：`ShopAdminLayout` 对 EDITOR 隐藏「品类管理」「官网留言」入口；品类页对 EDITOR 只读展示（若直接访问 URL）。

Server Action 层统一校验，UI 对 EDITOR 隐藏无权限入口（新建商品、列表上下架、删除、品类表单等）。
